# Preguntas y respuestas de algoritmos de programación

### 1. ¿Qué es un algoritmo en programación?
- A) Un programa completo
- B) Una secuencia finita de pasos para resolver un problema
- C) Un lenguaje de programación
- D) Un ciclo infinito
- **Respuesta correcta:** B  
- **Explicación:** Un algoritmo es una serie de instrucciones bien definidas que conducen a la solución de un problema en un tiempo finito.

### 2. ¿Qué tipo de estructura de control se utiliza para repetir una acción varias veces?
- A) Secuencia
- B) Condicional
- C) Bucle
- D) Función recursiva
- **Respuesta correcta:** C  
- **Explicación:** Un bucle permite repetir una serie de instrucciones varias veces.

### 3. ¿Qué algoritmo de ordenamiento tiene un mejor rendimiento en promedio?
- A) Burbuja
- B) Selección
- C) QuickSort
- D) Inserción
- **Respuesta correcta:** C  
- **Explicación:** QuickSort tiene un rendimiento promedio de \(O(n \log n)\), lo que es más eficiente que los otros algoritmos en promedio.

### 4. ¿Qué algoritmo utiliza una estructura de árbol binario para organizar datos de manera eficiente?
- A) Ordenamiento por inserción
- B) Búsqueda binaria
- C) Árbol binario de búsqueda
- D) Ordenamiento por burbuja
- **Respuesta correcta:** C  
- **Explicación:** Un árbol binario de búsqueda organiza datos para realizar búsquedas y ordenamientos eficientemente.

### 5. ¿Cuál es la complejidad temporal del algoritmo de búsqueda lineal?
- A) \(O(\log n)\)
- B) \(O(n)\)
- C) \(O(n^2)\)
- D) \(O(1)\)
- **Respuesta correcta:** B  
- **Explicación:** En la búsqueda lineal, cada elemento se revisa secuencialmente, por lo que su complejidad es \(O(n)\).

### 6. ¿Qué es un algoritmo recursivo?
- A) Un algoritmo que nunca se detiene
- B) Un algoritmo que se llama a sí mismo
- C) Un algoritmo que funciona en paralelo
- D) Un algoritmo que utiliza bucles
- **Respuesta correcta:** B  
- **Explicación:** Un algoritmo recursivo es aquel que se llama a sí mismo para resolver subproblemas más pequeños.

### 7. ¿Cuál es el mejor caso para el algoritmo de ordenamiento burbuja?
- A) Cuando la lista está desordenada
- B) Cuando la lista tiene un solo elemento
- C) Cuando la lista está completamente ordenada
- D) No tiene un mejor caso
- **Respuesta correcta:** C  
- **Explicación:** En el mejor caso, cuando la lista ya está ordenada, el algoritmo de burbuja sólo necesita una pasada.

### 8. ¿Cuál es la principal ventaja del algoritmo de búsqueda binaria sobre la búsqueda lineal?
- A) Puede trabajar con cualquier tipo de datos
- B) Funciona en tiempo constante
- C) Requiere que la lista esté ordenada, pero es mucho más rápida
- D) Funciona mejor en listas desordenadas
- **Respuesta correcta:** C  
- **Explicación:** La búsqueda binaria es mucho más rápida, con complejidad \(O(\log n)\), pero solo funciona en listas ordenadas.

### 9. ¿Qué problema resuelve el algoritmo de Dijkstra?
- A) Ordenar una lista de números
- B) Encontrar el camino más corto entre dos nodos en un gráfico
- C) Encontrar el número mayor en una lista
- D) Verificar si un número es primo
- **Respuesta correcta:** B  
- **Explicación:** El algoritmo de Dijkstra encuentra el camino más corto desde un nodo de origen a otros nodos en un grafo ponderado.

### 10. ¿Qué estructura de datos es comúnmente utilizada en un algoritmo de búsqueda en anchura (BFS)?
- A) Pila
- B) Cola
- C) Lista
- D) Árbol
- **Respuesta correcta:** B  
- **Explicación:** El algoritmo BFS utiliza una cola para explorar los nodos nivel por nivel.

### 11. ¿Cuál es la complejidad del algoritmo de ordenamiento por selección en el peor caso?
- A) \(O(\log n)\)
- B) \(O(n)\)
- C) \(O(n^2)\)
- D) \(O(n \log n)\)
- **Respuesta correcta:** C  
- **Explicación:** El ordenamiento por selección tiene una complejidad de \(O(n^2)\) en el peor caso, ya que compara cada elemento con todos los demás.

### 12. ¿Cuál es una desventaja del algoritmo de fuerza bruta?
- A) Es siempre el más eficiente
- B) Requiere una estructura de datos avanzada
- C) Puede ser ineficiente en términos de tiempo
- D) Solo funciona para problemas numéricos
- **Respuesta correcta:** C  
- **Explicación:** Los algoritmos de fuerza bruta tienden a ser ineficientes ya que prueban todas las posibles soluciones.

### 13. ¿Qué es la complejidad espacial de un algoritmo?
- A) El tiempo que tarda en ejecutarse
- B) La cantidad de memoria que usa
- C) La cantidad de datos que puede procesar
- D) El número de veces que se ejecuta
- **Respuesta correcta:** B  
- **Explicación:** La complejidad espacial se refiere a la cantidad de memoria que requiere un algoritmo.

### 14. ¿Cuál es la complejidad de una búsqueda en un árbol binario balanceado?
- A) \(O(n)\)
- B) \(O(\log n)\)
- C) \(O(n^2)\)
- D) \(O(1)\)
- **Respuesta correcta:** B  
- **Explicación:** En un árbol binario balanceado, la búsqueda tiene una complejidad de \(O(\log n)\), ya que divide el problema a la mitad en cada paso.

### 15. ¿Qué problema resuelve el algoritmo de Floyd-Warshall?
- A) Ordenar elementos en un array
- B) Encontrar los caminos más cortos entre todos los pares de nodos en un grafo
- C) Encontrar un ciclo en un grafo
- D) Verificar si un número es primo
- **Respuesta correcta:** B  
- **Explicación:** Floyd-Warshall es un algoritmo que resuelve el problema de encontrar los caminos más cortos entre todos los pares de nodos en un grafo.

### 16. ¿Qué tipo de algoritmo es el MergeSort?
- A) Algoritmo iterativo
- B) Algoritmo de divide y vencerás
- C) Algoritmo de búsqueda
- D) Algoritmo de fuerza bruta
- **Respuesta correcta:** B  
- **Explicación:** MergeSort es un algoritmo de "divide y vencerás", ya que divide el problema en partes más pequeñas y las resuelve recursivamente.

### 17. ¿Qué algoritmo de ordenamiento es el más adecuado para listas pequeñas?
- A) QuickSort
- B) Ordenamiento por burbuja
- C) Ordenamiento por selección
- D) Inserción
- **Respuesta correcta:** D  
- **Explicación:** El algoritmo de inserción es eficiente para listas pequeñas, con complejidad \(O(n^2)\) en el peor caso, pero es sencillo y rápido para pocos elementos.

### 18. ¿Qué algoritmo de búsqueda es más eficiente en un array ya ordenado?
- A) Búsqueda lineal
- B) Búsqueda binaria
- C) Búsqueda por interpolación
- D) Búsqueda en profundidad
- **Respuesta correcta:** B  
- **Explicación:** La búsqueda binaria es más eficiente en un array ordenado con complejidad \(O(\log n)\).

### 19. ¿Qué característica define a un algoritmo codicioso (greedy)?
- A) Intenta encontrar la solución óptima a través de retroceso
- B) Selecciona la mejor opción local en cada paso con la esperanza de encontrar la solución global óptima
- C) Divide el problema en subproblemas más pequeños
- D) Siempre encuentra la solución global óptima
- **Respuesta correcta:** B  
- **Explicación:** Un algoritmo codicioso selecciona la mejor opción localmente en cada paso con la esperanza de llegar a la mejor solución global.

### 20. ¿Qué significa que un algoritmo tenga complejidad \(O(1)\)?
- A) Su tiempo de ejecución es constante
- B) Se ejecuta en tiempo exponencial
- C) Su tiempo de ejecución depende del tamaño de la entrada
- D) Es un algoritmo recursivo
- **Respuesta correcta:** A  
- **Explicación:** Su tiempo de ejecución es constante, sin importar el tamaño de la entrada.

# Preguntas sobre conceptos de programación

### 1. ¿Qué es una variable en programación?
- A) Un valor constante que no puede cambiar.
- B) Un espacio en memoria para almacenar datos.
- C) Una función que realiza una operación.
- D) Un error en el código.
- **Respuesta correcta:** B

### 2. ¿Qué es una función en programación?
- A) Un tipo de variable.
- B) Una estructura de datos.
- C) Un bloque de código reutilizable que realiza una tarea específica.
- D) Una iteración dentro de un ciclo.
- **Respuesta correcta:** C

### 3. ¿Qué significa el término "bucles" en programación?
- A) La repetición de un conjunto de instrucciones un número finito de veces.
- B) Un tipo de variable que almacena datos.
- C) Un proceso que detiene la ejecución del programa.
- D) Un tipo de función matemática.
- **Respuesta correcta:** A

### 4. ¿Qué es la recursión en programación?
- A) Un proceso que involucra la repetición infinita.
- B) Una técnica que permite a una función llamarse a sí misma.
- C) Un error de compilación.
- D) Una forma de realizar cálculos matemáticos complejos.
- **Respuesta correcta:** B

### 5. ¿Qué es un "array" o "arreglo" en programación?
- A) Un conjunto de variables de diferente tipo.
- B) Un conjunto de valores almacenados en posiciones consecutivas de memoria.
- C) Un valor constante que no cambia.
- D) Una función que devuelve un valor.
- **Respuesta correcta:** B

### 6. ¿Qué es un condicional "if" en programación?
- A) Una función que detiene la ejecución del programa.
- B) Un bucle que se repite infinitamente.
- C) Una declaración que permite ejecutar un bloque de código solo si se cumple una condición.
- D) Un tipo de dato que almacena números enteros.
- **Respuesta correcta:** C

### 7. ¿Qué es un compilador?
- A) Un programa que convierte el código fuente en código ejecutable.
- B) Un tipo de variable que almacena funciones.
- C) Un proceso que guarda datos en la memoria.
- D) Un algoritmo de búsqueda.
- **Respuesta correcta:** A

### 8. ¿Qué significa el término "tipo de datos" en programación?
- A) Una función que realiza operaciones matemáticas.
- B) Un formato que define el tipo de valor que puede almacenar una variable.
- C) Una instrucción que inicia un bucle.
- D) Una estructura de control condicional.
- **Respuesta correcta:** B

### 9. ¿Qué es la orientación a objetos en programación?
- A) Un paradigma que organiza el software en bloques llamados funciones.
- B) Un enfoque de programación donde los datos y las funciones relacionadas se agrupan en objetos.
- C) Un tipo de bucle que permite la recursión.
- D) Un algoritmo de ordenamiento.
- **Respuesta correcta:** B

### 10. ¿Qué es el depurador (debugger) en programación?
- A) Un programa que detecta y corrige errores automáticamente.
- B) Una herramienta que permite examinar el código para identificar y corregir errores.
- C) Un tipo de compilador.
- D) Un lenguaje de programación.
- **Respuesta correcta:** B
